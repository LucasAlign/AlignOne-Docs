# Families Served Over Time

**Who this is for:** Program staff (Admins and Coordinators).
**What it is:** A chart on the **Reports & Impact** page.

## What it shows

This area chart shows how the number of families you serve has changed across recent months. Each point on the chart represents one month, so you can see growth and trends at a glance rather than a single point-in-time number.

## Options you can change

You reach these via the gear (⚙) on the panel while in Customize mode.

- **Cumulative total** — shows a running total, so each month includes every family added up to that point.
- **New per period** — shows only the families added in each month, so you can compare month-to-month growth.

## Which filters affect it

All three filters apply: **Date range**, **County**, and **Church**. The **Date range** filter sets how many months the chart shows, so a wider range stretches the chart back further in time. **County** and **Church** narrow the chart to the families that match.

## Tips & notes

!!! tip "Pick the view that answers your question"
    Use **Cumulative total** to show overall reach over time, and **New per period** when you want to highlight how many families joined each month.
