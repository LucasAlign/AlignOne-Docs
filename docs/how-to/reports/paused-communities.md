# Paused Communities

**Who this is for:** Program staff (Admins and Coordinators).
**What it is:** A counter tile on the **Reports & Impact** page.

## What it shows

This tile counts the families whose status is paused — care communities that are on hold rather than actively being served. By default it counts families marked **Paused**.

## Options you can change

While in Customize mode, select the **gear (⚙)** on the panel to change what gets counted:

- **Paused statuses** — use the checkboxes to choose which statuses count as "paused." Check or uncheck statuses so the tile matches how your program defines a paused community.

## Which filters affect it

- **County** and **Church** apply — they narrow the tile to the families in that county or served by that church.
- **Date range does not apply.** The tile reflects each family's current status, and the app doesn't keep a history of when a status changed, so there's no past window to count.

## Tips & notes

!!! note "Current count only"
    How long a community has been paused, and why, aren't shown yet — the tile reports only the current number of paused communities.
